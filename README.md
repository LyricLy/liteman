# ltman


## Requirements

- Python 3.8.3+
- ffmpeg

## Installation

Ensure that `ffmpeg` is included in your PATH environment variable.

- `git clone https://github.com/tylergibbs2/ltman`
- `cd ltman`
- `python -m pip install -r requirements.txt`

## Usage

```py
python -m ltman https://someurlhere/
```
